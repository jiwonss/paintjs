# paintjs
Painting Board made with VanilaJS

Painting Board
==============

노마드코더 - 바닐라 JS로 그림판 만들기

기능
----

1. Change Color

    색 선택

2. Brush Size
  
    브러시 사이즈 변경

3. Paint/Fill
  
    Paint - 그림 그리기
  
    Fill - 색 채우기

4. Save
  
    저장
